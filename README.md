# Weather App (API Integration)

This project fetches real-time weather data using an API.

## Features
- Input city name
- Displays temperature (in Celsius)
- Displays humidity
- Shows weather condition
- Handles invalid city names

## Technologies Used
- Python
- requests library
- OpenWeatherMap API

## How to Run
1. Install requests library (pip install requests)
2. Add your API key in the code
3. Run the Python file
4. Enter city name to get weather details
